★ ACTIVE
Active Workout — interactive

Overview · set-logging · summary. Built in React, full behavior — recreate natively.

Active Workout · behavior rules

  • Auto-finish: both kg + reps filled and field blurs → set marks done.
  • Auto rest: completing a set → 60s rest, unless final set. Completing exercise → 90s. Adding an empty set also starts it.
  • Rest pill: draggable anywhere (position persists), ring countdown, +15s, pause, dismiss.
  • Complete Exercise: drops empty sets, marks filled done, celebration, 90s rest, back to overview.
  • Celebrations: weight > PR → "New PR!"; reps > target → "Extra reps!"; finish → summary.
  • No commit until Save: edits held live; scope dialog = today-only vs permanent (My Plans copy, never predefined program).
  • ⋯ menu: substitute / add / remove / create superset. End sheet: Finish & Save · Discard · Continue.
